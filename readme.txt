== Description ==

Velox is a clean, minimal personal/portfolio WordPress block theme. Under the hood it is architected generically — no assumptions about content type or use case. Adding a new site type means adding templates and patterns, not rewriting core files.

All design decisions live in theme.json. Swapping the entire look means editing one file. Built for WordPress 6.6+ with full block editing support.
